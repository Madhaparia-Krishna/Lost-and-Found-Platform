Lost & Found System Email Templates

How to use these templates: Copy the HTML code from each template and paste it into your EmailJS email template editor. The parameter placeholders (like {{user_name}}) will be replaced with actual values when the email is sent.

2. Account Blocked Email

Required Parameters:
  • to_email: Recipient's email address
  • user_name: User's name
  • block_reason: Reason for blocking (optional)
  • block_date: Date when account was blocked

            color: white;
            padding: 20px;
            text-align: center;
        }
        .email-body {
            padding: 20px;
        }
        .email-footer {
            background-color: #f7f7f7;
            padding: 15px;
            text-align: center;
            font-size: 12px;

        }

            background-color: #3498db;
            color: white;

4. Match Found Email (For Reference)

Required Parameters:
  • to_email: Recipient's email address
  • user_name: User's name
  • item_title: Title of the matched item
  • category: Category of the item
  • message: Custom message about the match
  • match_link: Link to view the match details
  • location: Location where item was found
  • time: Time of match notification
  • date: Date when item was found
